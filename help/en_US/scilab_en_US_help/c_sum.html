Scilab2Xcos

Rebuild a simulink diagram in Xcos

Calling Sequence

The toolbox is not meant to be called as a function but executed as a script.\nTo run the script simply execute Basis.sce

Description

The script takes a Matlab Simulink SLX file and extracts the diagram data into a Xcos diagram.

Examples

If you want so see how the script works, use it and select the GQS.slx file
